**Educational async brute-force auditor - 10x faster than Hydra**

## ⚡ Features

- 🚀 **Async I/O** with Tokio (10,000+ concurrent attempts)
- 🔌 **Multi-protocol**: SSH, HTTP/HTTPS, FTP (more coming)
- 🌐 **Tor integration** for anonymity
- 🐍 **Python plugin system** - write custom modules
- 🔄 **Proxy rotation** (SOCKS5, HTTP)
- 📊 **Real-time progress** with pretty UI
- ⚙️ **Highly configurable**
